# Frontend — Sistema de Gestión de Tareas y KPIs (BeZeta)

React + TypeScript + Vite. Ver instrucciones de instalación y arquitectura general en el
[README del repositorio](../README.md).

## Comandos

```powershell
npm install
npm run dev        # servidor de desarrollo (http://localhost:5173)
npm run build       # typecheck + build de producción
npm run lint
```
